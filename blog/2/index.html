Is HTML really a programming language?!

5/23/2022 | ...sec read

TLDR; No, HTML is not a programming language.

So you may know what HTMl is? It is a programming language to organize your text. Like adding context and structure. Well what is programming then? "a programming language allows you to describe some sort of process of doing something, " Thanks to stack overflow for that definition.

Since html is a way of adding context and structure to text it is not a offical coding language Don't be bummed though, more complex things involving html like Javascript, CSS, and Ajax then those are considered a offical programming language
